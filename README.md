# 🚀 cryptomania-app

**cryptomania-app** adalah aplikasi web yang menyediakan informasi dan analisis terkini tentang cryptocurrency. Aplikasi ini dirancang untuk membantu pengguna memahami tren pasar cryptocurrency melalui data real-time, grafik interaktif, dan fitur pencarian yang mudah digunakan.

---

## ✨ Fitur Utama

- Menampilkan daftar cryptocurrency populer dengan harga terbaru secara real-time
- Grafik interaktif untuk melihat tren harga cryptocurrency dalam berbagai periode waktu
- Informasi detail tentang masing-masing cryptocurrency, termasuk kapitalisasi pasar, volume perdagangan, dan perubahan harga
- Fitur pencarian dan filter untuk menemukan cryptocurrency tertentu dengan cepat
- Antarmuka yang responsif dan ramah pengguna, dapat diakses dari berbagai perangkat

---

## 🛠️ Instalasi

Ikuti langkah-langkah berikut untuk menjalankan aplikasi ini secara lokal di komputer Anda:

```bash
# Clone repository
git clone https://github.com/HafiyahPutra/cryptomania-app.git

# Masuk ke direktori proyek
cd cryptomania-app

# Install dependencies menggunakan npm
npm install

# Jalankan aplikasi
npm start
```

> **Catatan:** Pastikan Anda sudah menginstal [Node.js](https://nodejs.org/) dan npm sebelum menjalankan perintah di atas.

---

## 📋 Penggunaan

Setelah aplikasi berjalan, buka browser dan akses alamat berikut:

```
http://localhost:3000
```

Anda dapat menggunakan fitur pencarian untuk menemukan cryptocurrency tertentu, klik pada nama koin untuk melihat detail dan grafik harga yang interaktif.

---

## 🧰 Teknologi yang Digunakan

| Teknologi       | Keterangan                            |
|-----------------|---------------------------------------|
| React.js        | Library JavaScript untuk membangun UI |
| Axios           | Untuk melakukan HTTP request API      |
| Chart.js        | Untuk membuat grafik interaktif       |
| CSS / SASS      | Styling dan tata letak aplikasi       |
| CoinGecko API   | Sumber data harga dan informasi crypto|
| NewsAPI         | Sumber berita dan informasi crypto    |
| VercelApp       | Deploy Website dan NewsAPI            |
| Render          | Deploy AI LSTM Prediksi harga         |
